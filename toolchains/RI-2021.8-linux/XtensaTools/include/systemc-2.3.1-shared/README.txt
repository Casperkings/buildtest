These header files from the OSCI SystemC 2.3.1 distribution have been modified to
support xtsc-run plugins which requires using a shared library version of the
SystemC library (rather then a static library version).

Modifications are required to support Microsoft Visual C++ coding procedures for
exporting and importing symbols from a DLL.
